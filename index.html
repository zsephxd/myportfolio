This is our capstone project, completed as a team of three. I played a key role as the web developer, responsible for building a website using native PHP, HTML/CSS, and Bootstrap that functioned as the central database for the system. All data collected from the Android mobile application was stored on the website, and I handled the communication between the website and the Android app, ensuring seamless data requests and transfers.

I developed a data map for both the website and the Android application to visually display the distribution of corn diseases across different areas. In addition to my web development tasks, I contributed to the Android application by developing the image processing model using Python, achieving a 92% accuracy in classifying corn diseases.

The entire project was completed at zero cost, showcasing our ability to deliver a functional, cost-effective solution.

The Llanera Teachers Association Multi-Purpose Cooperative (LTAMPC) is a small association that offers services such as loans, savings, and other financial transactions. During my OJT as a full-stack developer, I noticed their manual process for managing member records and transactions was time-consuming.

To streamline their workflow, I developed a web-based profiling system using native PHP, HTML/CSS, and Bootstrap. This system allowed the staff to quickly search for and manage member information, saving significant time compared to their manual processes. I also implemented an automatic age calculation feature to ensure member ages were always current.

Additionally, I created a function that allowed the cooperative to generate filtered search results into Excel files, making data extraction more efficient. I set up the system using XAMPP, enabling all computers on their local network to input or delete data from the database in real time. This solution improved the cooperative's daily operations, providing a cost-effective and user-friendly system.
